Sie haben eine Liste mit ganzzahligen Gewichten erhalten. Sie sollen Stephen helfen, diese Gewichte in zwei Gruppen aufzuteilen, und zwar so, dass die Differenz zwischen dem Gesamtgewicht der beiden Gruppen so gering wie möglich ist.

Eingabedaten: Eine Liste der Gewichte als Liste von Ganzzahlen.

Ausgabedaten: Die Zahl, die die kleinstmögliche Gewichtsdifferenz darstellt, als positive Ganzzahl.

Beispiel:

checkio([10,10]) == 0
checkio([10]) == 10
checkio([5, 8, 13, 27, 14]) == 3
checkio([5,5,6,5]) == 1
checkio([12, 30, 30, 32, 42, 49]) == 9
checkio([1, 1, 1, 3]) == 0

Wie es verwendet wird: Dies ist eine kombinatorische Optimierungsversion des Partitionsproblems. Die kombinatorische Optimierung ist weit verbreitet und Sie sehen sie oft, wenn Sie sich mit automatisierten Zeitplänen oder Routen berechnung.
